# ADR 0078: Micro-experts are advisory and authority-free

**Status:** Accepted  
**Date:** 2026-07-16

Routing, language, safety, and complexity micro-experts emit deterministic
fixed-point advice with reasons. Their contract cannot represent execution or
permission authority. Core permission, scheduler admission, verifier activation,
and result release remain independent mandatory gates. Safety non-matches are
not approvals. Package selection requires exact runtime/digest binding.
